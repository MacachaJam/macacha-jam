## Cositas

- Para los diálogos usa el addon [Dialogue Manager](https://github.com/nathanhoad/godot_dialogue_manager)
- Lo arrancamos usando la plantilla [bitbrain/godot-gamejam](https://github.com/bitbrain/godot-gamejam). Aunque después lo reemplazamos tanto que ya casi está erradicado del repo.
- Se exporta a github pages automágicamente gracias a [endlessm/amalgamate-pages](https://github.com/endlessm/amalgamate-pages)
- Y muchas cosas roba.. inspiradas en [endlessm/threadbare](https://github.com/endlessm/threadbare) como:
  - El scene manager: cambiar de escena con transiciones, teleporters, spawn points (convertido a 3D)
  - El game state basado en Resources
  - El area filler, convertido a 3D
